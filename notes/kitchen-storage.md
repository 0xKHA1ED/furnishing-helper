# Kitchen storage plan (≤ 25k EGP)

## Locked (2026-07-17)

| Item | Decision |
|------|----------|
| Budget | **Hard max 25,000 EGP** |
| Scope | **Storage only** (shelves, pantry, organizers, wall/freestanding units) |
| Kitchen state | Plumbing + electric only — **no cabinets/counters** |
| Shopping | **Best quality under budget** (Cairo / 6th of October); IKEA welcome but not required |
| Out of this bag | Fridge, washer, cooker (owned), water heater, full remodel |

## Lifestyle storage drivers

- Friday meal-prep + big shop every 1–2 weeks → dry goods + container storage  
- Freezer-heavy cooking → still need fridge-adjacent dry storage and prep organization  
- Already own cooker, air fryer, kettle, blender → leave clear appliance zones  

## Design approach (draft)

Empty kitchen → prioritize **freestanding / modular** systems that work without built-in carcasses (rent-friendly, movable), optionally add wall shelves if walls allow drilling.

### Priority stack under 25k

1. **Primary pantry / tall storage** (food + bulk)  
2. **Base / open storage for pots + appliances**  
3. **Wall shelves or rail systems** for daily grab items  
4. **Organizers** (bins, lids, under-shelf) last
